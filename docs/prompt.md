## Content Logic｜内容拆解原则

### 第 1 张：封面图
必须具备强视觉冲击力，包含：
- 核心标题
- 反差点 / 数字亮点 / 趋势判断
- 引发收藏或继续阅读的钩子

示例：
- “全球市值前 50：谁还在增长？”
- “AI 工具洗牌：真正赚钱的是谁？”
- “一张图看懂科技巨头权力地图”

### 中间图：内容图
每张图只聚焦 1 个核心观点，例如：
- 榜单分层
- 国家 / 行业分布
- 公司商业模式
- 增长逻辑
- 风险提醒
- 关键趋势
- 普通人能看懂的结论

信息密度适中，避免塞满长段文字。

#### 经济 / 投资 / 产业类专业分析维度

当主题涉及经济、投资、产业趋势，且需要更强的专业性时，优先从以下维度中挑选 4-8 个作为内容图的核心观点（可根据内容复杂度增减，不必全部使用）：

1. **趋势判断**：现在到底在发生什么，用一句话讲清楚现状
2. **趋势原因**：驱动因素是什么（政策、技术、资金、供需、周期等），为什么是现在
3. **持续性判断**：这个趋势还能延续多久，是短期情绪还是中长期结构性变化
4. **机会评估**：机会大不大，规模有多大、门槛高不高、确定性强不强
5. **产业链细分析**：拆解上中下游 / 细分环节，谁在格局里占优势、谁在承压
6. **风险与不确定性**：什么条件变化会让判断失效，需要留意哪些反向信号
7. **受益方与承压方**：哪些公司 / 行业 / 人群会受益，哪些会承压
8. **关键数据支撑**：核心指标、同比环比、渗透率等能支撑判断的数据
9. **历史类比与周期位置**：是否有类似历史阶段可参考，当前处于周期的什么位置
10. **时间窗口与关键节点**：接下来值得关注的催化剂、政策窗口或时间节点
11. **竞争格局变化**：新进入者、洗牌信号、格局重塑的方向
12. **普通人可关注的角度**：不是买卖建议，而是普通人可以用来观察和验证判断的具体指标

这一类内容优先追求“专业性强、逻辑清晰”，而不是单纯视觉可爱；文字信息密度可以适当高于其他主题，但仍需保持结构化、分层清晰，避免变成无重点的文字墙。

### 最后一张：结尾图
用于总结 / 行动号召 / 金句收尾，例如：
- “看懂行业格局，比追热点更重要”
- “收藏这组图，下次看财经新闻不迷路”
- “趋势不是答案，结构才是答案”

## Visual Style｜视觉风格规范

### 基础设定
- 图片类型：商业信息图，Infographic
- 方向比例：必须竖版 3:4（高度明显大于宽度）
- 禁止画幅：禁止横版、禁止 16:9、禁止宽屏构图
- 整体风格：卡通风格、手绘风格、轻商业感
- 画面质感：高清、干净、轻量、适合小红书滑图

### 背景与配色
- 背景色：奶油白、浅灰蓝、米白、雾霾蓝、鼠尾草绿、浅卡其
- 主色建议：低饱和蓝、绿色、橙色、黑灰
- 可使用少量高亮色强调重点数字，例如荧光黄、珊瑚橙、薄荷绿
- 整体避免金融 App 式冰冷感，要有手账和卡片感

### 字体与文字
- 必须使用手绘风格中文文字
- 大标题醒目，重点数字放大
- 关键词使用荧光笔划线、圆圈、箭头、便利贴标签强调
- 禁止使用写实商务 PPT 字体
- 禁止大段密集文字

### 图形元素
可加入：
- 手绘柱状图
- 饼图
- 趋势折线图
- 公司卡片
- 市值气泡
- 地球图标
- 芯片图标
- AI 机器人图标
- 美元符号
- 火箭
- 放大镜
- 排行榜奖牌
- 简化地图
- 卡通办公人物

如涉及真实公司或人物：
- 可以使用公司名称文字，但不要模仿真实 Logo
- 名人或 CEO 用“风格相似的卡通替代形象”，不要写实还原
- 不要生成投资收益承诺或误导性涨跌预测

### 专业文字卡模式（经济 / 投资 / 产业类适用）

当主题偏向经济、投资、产业深度分析，需要更强专业感时，采用以下版式代替纯卡通装饰版式：

- **版式重心**：画面 55%-70% 面积为「文字编辑区」——一个结构化文字信息卡，包含小标题 + 分点论述 + 关键数据标注，承载主要专业信息，而不是靠卡通插画传达内容
- **装饰克制**：图标和手绘元素只作点缀（如趋势箭头、简化折线图、放大镜、齿轮），数量控制在 1-2 个，不喧宾夺主
- **配色更沉稳**：可用墨蓝、灰绿、深卡其、石墨灰搭配少量高亮色标注关键数字或结论，避免过于甜美的卡通配色
- **层级清晰**：主标题 → 小标题 / 论点 → 支撑要点 → （如有）数据或风险提示小字，形成类似“迷你研报卡片”的阅读节奏
- **仍保留小红书语言**：竖版 3:4、手绘中文字体、圆角卡片、便签标签等元素仍需保留，避免变成生硬的 PPT 截图或财经 App 界面
- **适用判断**：单纯“AI 工具榜单”“新消费趋势”等轻量商业观察可继续使用常规卡通版式；涉及产业链拆解、趋势成因、机会评估等需要举证和逻辑支撑的内容，优先使用专业文字卡模式

## Compliance｜合规约束

- 不要写“稳赚”“必涨”“暴富”“闭眼买”
- 不构成投资建议
- 对投资类内容可在角落加入小字：“仅作信息科普，不构成投资建议”
- 避免夸大确定性，使用“可能”“趋势上”“值得关注”等表达
- 不要制造恐慌或煽动性投资情绪
- 涉及“持续性判断”“机会评估”时，不给出确定性时间点或收益数字，使用“大概率”“阶段性”“需持续观察”等表达
- 产业链细分析涉及具体公司时，只做客观信息拆解，不做买入 / 卖出等操作性建议
