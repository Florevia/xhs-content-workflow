# 小红书内容合规自查

1. 不写虚假亲测。
2. 不写无法证明的绝对化承诺。
3. 不写“100%有效”“全网第一”“永久解决”等绝对词。
4. 涉及医疗、保健、金融、法律等内容必须加免责声明。
5. 商业合作、广告、赠品体验要按平台和法律要求披露。
6. 不搬运他人图片、文案、评论。
7. 不诱导用户违规私信、站外交易或刷量。
8. 不做批量低质重复内容。
9. 不自动登录、不读取 Cookie、不绕过验证码、不模拟无人值守批量发布。
10. 小红书正文字数上限为 1000 字（含文末话题标签）；生成正文时请控制篇幅，避免发布时因超长失败。
11. **金融内容（银行、证券、基金、新股/打新等）额外红线**：该条适用于笔记、评论、用户资料等小红书所有发布场景，不限于笔记正文。
    - 禁止在无资质情况下提供个股/新股分析、收益承诺或收益预测（如具体中签率数字、具体收益率、"稳赚""必中""闭眼打新"等绝对化表述）。
    - 禁止冒充专业人士（如自称持牌顾问/分析师）或暗示内容代表持牌机构意见。
    - 禁止推广非法集资、传销活动。
    - 禁止发布违规借贷/高利贷信息；讲解融资认购（如孖展倍数、杠杆比例）等操作性细节时只做机制层面科普，不给出具体操作参数或加杠杆建议，统一引导"通过持牌机构了解完整条款和风险揭示"。
    - 金融类笔记必须保留"不构成投资建议、不代表持牌机构意见"等免责声明，不得为追求简洁而删除。
    - 若收到平台针对金融内容的合规提醒/处罚通知，应在核实违规点后编辑对应笔记修订内容，无需重新发布；修订应优先移除通知中提示的风险表述类型（个股分析/收益承诺/专业人士包装/非法集资/传销/违规借贷等），而非逐字对照通知模板。
